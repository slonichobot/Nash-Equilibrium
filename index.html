Lemke-Hawson algorithm

Following table shows the steps of Lemke-Hawson algorithm for each possible starting configuration. The state of the algorithm is determined by a pair of vertices, one from each polyhedron. The algorithm starts in the pair of vertices (0,0) (which is fully labeled) and drops any of the labels. This is equivalent to giving the corresponding pure strategy a positive probability. The drop of a label can be interpreted as a traverse through an edge of corresponding polyhedron and the dropped label corresponds to the inequality which seized to be bounding. When arriving to a new vertex, a new bounding inequality appeared, so a new label was discovered.
